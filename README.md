# Django-saas-broilerplate

A Django SaaS boilerplate:
- Multi-tenant (django-tenants).
- CMS: Wagtail (pages, blogs, SEO).
- Payments: Stripe + Paystack (subscriptions).
- Themes: Tailwind CSS, Unfold admin, switchable.
- Landing/blog: responsive, sitemap.xml.
- Deploy-ready (Vercel, Postgres).
- Extras: auth, API (DRF), email, docs + setup guild.
